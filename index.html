Projects

AXI Protocol Memory Controller

Designed AXI Designed AXI Master and Slave RTL modules in Verilog for high-performance SoC integration. Built a SystemVerilog-based testbench with drivers, monitors, scoreboards, and assertions to ensure AXI protocol compliance. Used QuestaSim 10.7c for simulation and debug, executing diverse test scenarios and validating design correctness. Authored a comprehensive Verification Plan Document (VPD) to structure the verification process.

APB IP Core Verification

Verified the APB finite state machine, covering idle, setup, and access states in accordance with the AMBA APB protocol specification. Developed a class-based UVM testbench with sequencers, drivers, monitors, and a scoreboard to ensure full compliance. Utilized SystemVerilog assertions and functional coverage models to achieve 100% code and functional coverage, ensuring thorough verification of protocol behavior and FSM transitions.

PCIe Design and Verification SerDes Block

Designed and verified a PCIe Gen2 SerDes block using Verilog for RTL implementation and SystemVerilog for verification. Developed core modules including Scrambler, Descrambler, 8b/10b Encoder/Decoder, and parallel-to-serial (PISO)/serial-to-parallel (SIPO) converters. Constructed a UVM-based testbench with layered components to verify data integrity and protocol compliance, achieving complete functional coverage. Conducted in-depth analysis of LTSSM behavior, PIPE interface signaling, Data Link Layer Packets (DLLPs), and 8b/10b encoding schemes to validate end-to-end SerDes functionality.
